## Instruções 

> Rode o comando em seu terminal 

```
git clone git@github.com:JoaoVFerreira/Project-Starwars.git
```

> Entre na pasta do projeto 

```
cd Project-Starwars
```

> Instale as dependências

```
npm install
```

> Rode o comando npm start

```
npm start
```
